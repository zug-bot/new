# Sprites Directory

Add your 2.5D isometric character and enemy sprites here.
- Use PNG format with transparency.
- Style: Pixel art, isometric, inspired by Triangle Strategy and Octopath Traveler.
- Lighting: You may add "glow" or "highlight" layers for lighting effects.
- Shadows: Add shadow blobs or semi-transparent ellipses under sprites.

Example files:
- hero_idle.png
- hero_attack.png
- enemy_goblin.png

You can find placeholder or free assets at:
- https://itch.io/game-assets/tag-isometric
- https://opengameart.org/
- https://kenney.nl/assets?q=isometric

Be sure to respect licensing requirements.
